# adctf-2025 Challenges by Phrinky

本仓库为 Phrinky 在 adctf-2025 所出题的题目仓库

## 仓库结构

`challenges` 目录下每个目录名字为每道题的 **标识符**，每个目录下包含以下子目录和文件:

- `source`: 源代码
- `writeup`: 解题思路
- `README.md`: 题目描述

`scripts` 文件夹包含用于推送及处理仓库用的脚本。
